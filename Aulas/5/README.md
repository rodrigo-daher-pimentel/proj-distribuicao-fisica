# Aula 5 — Acompanhamento Sprint 1

## Objetivo

Aula de acompanhamento para alinhar os experimentos computacionais que cada equipe deve realizar antes da Sprint Review #1.

## Experimentos definidos

### Obrigatorios (todos)

1. Teste com instancias C1–C4 (aumento do numero de clientes)
2. Solver utilizado
3. Limite de tempo
4. Parametros de frota
5. Qualidade da solucao obtida
6. Utilizacao e remocao da formulacao MTZ

### Opcionais (cada equipe escolhe pelo menos 1)

- Variacao do custo fixo do VUC
- Variacao da velocidade media
- Variacao do tempo de servico
- Analise do gap x time limit (30s, 60s, 300s)
- Comparacao com outras equipes da turma
